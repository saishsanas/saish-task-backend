# SaishTask

SaishTask is a full-stack task management application designed to organize tasks efficiently.
It includes a Spring Boot backend and a React (Vite + TypeScript) frontend, featuring real-time task list updates and a clean, responsive UI.

---

## Tech Stack

### **Backend**
- Java 21
- Spring Boot
- JPA / Hibernate
- **PostgreSQL (Docker)**
- Docker & Docker Compose
- Maven

### **Frontend**
- React + Vite
- TypeScript
- Axios
- Tailwind CSS / NextUI Components

---

##  Features

- Create, update, delete task lists
- Add tasks inside a task list
- Progress bar for task completion
- Responsive UI design
- Full CRUD REST API
- Clean architecture pattern
    - Controller → Service → Repository
    - DTOs & Mappers
- Global exception handling
